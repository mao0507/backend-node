# backend-node

## 簡介

建立一個通用的node js 後端 API 系統，將大概率會使用到的項目先整理寫好

## 使用方法 

1. 安裝依賴

```
$ yarn 
```

2. 執行程式
```
$ yarn dev  //本機啟動
$ yarn serve //正常啟動，不使用nodemon
```

## 使用套件

1. yarn 
2. dotenv
3. express
4. http
5. request
6. helmet
7. jsonwebtoken
8. body-parser
9. nodemon
10. swagger-autogen
11. swagger-jsdoc
12. swagger-ui-express
13. knex
14. mysql2
